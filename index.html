📖 手相怎么看？点击展开说明

看手相分两步：先看三大主线，再看七大掌丘。

🖐 三大主线（右手掌）
  • 生命线：环绕拇指根部 → 看体质、精力、寿命
  • 智慧线：横穿掌心 → 看思维、判断力、学习能力
  • 感情线：小指下方向食指 → 看情感模式、婚姻
⛰ 七大掌丘位置

掌丘饱满 = 该方面运势强；扁平 = 较弱

木星丘（食指根）= 权力 · 土星丘（中指根）= 责任 · 水星丘（无名指根）= 财运 · 金星丘（拇指根）= 爱情 · 月丘（掌缘）= 直觉 · 火星丘 = 勇气 · 地球丘/海王星丘 = 根基

🏠 九宫方位图（房屋平面）
⭐ 九星含义
  • 🟢 一白水：桃花、人缘（吉）
  • 🟤 二黑土：病符、是非（凶）
  • 🔵 三碧木：口舌、争斗（凶）
  • 🟢 四绿木：文昌、学业（吉）
  • 🟡 五黄土：大凶灾星（大凶）

  • 🔴 七赤金：破财、盗贼（凶）
  • ⚪ 八白土：正财、置业（大吉）

用法：输入年份 → 查看飞星图 → 吉星方位（绿/白/紫）宜常待、宜开门窗；凶星方位（黄/黑/赤）宜避开或放化煞物（如金属物泄土、水泄火）。

📛 姓名学分析（三才五格）

📖 姓名学怎么看？点击展开说明

姓名学用康熙字典笔画计算五个"格"，每个格对应人生不同阶段：

格算法代表
天格姓笔画 + 1祖先、长辈、先天运势
人格姓笔画 + 名第一字笔画本人、主运（最重要）
地格名字两字笔画之和（单名+1）子女、晚辈、36岁前运
外格总格 - 人格 + 1社交、外界评价
总格全部笔画之和一生总运、36岁后

五格数字对应 81 数理，每格分为 🟢吉 / 🟡半吉 / 🔴凶。三才（天·人·地）的五行配置影响整体评分。

用法：输入中文姓名 → 点「分析」→ 看五格数理和三才配置。

📛 示例：「李白」姓名分析结果
李白 · 中吉 (72分)
天格 8画 中吉 · 天格
祖运、长辈运、上司运
意志坚定，勤勉发展。
人格 18画 中吉 · 人格
主运、本人、婚姻
权威显达，博得名利，且养柔德。
地格 6画 大吉 · 地格
前运、子女运（36岁前）
天德地祥俱备，福庆甚丰。
总格 18画 中吉 · 总格
总运、晚年运（36岁后）
有志竟成，内外有运。
三才配置： 金·金·土 → 三才顺遂，基础稳固

以上分析基于三才五格法，仅供参考。笔画为估算值，精确分析需查康熙字典。
